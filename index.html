CollectorsDojo.com

This is a passion project that I built using Laravel. It allows the user to search for and research video games from the past and present. The games are stored in an data structure that I built using Eloquent API Resources. The games range back over 30 years and come from an assortment of over 30 consoles. The users are able to build out and store their own collection using the app with very specific details about each game including purchase date, price, and location.

ReacTrivia.com

This is a Single-Paged App that I built using React.js with Redux and React Router. The application is a trivia game, where the user can select betweeen the categories of Movies, Sports, or Places. The user will have 60 seconds per round, and they will answer the questions in a multiple choice format. The player is rewarded 1 point for every correct answer and penalized 1 point for every incorrect answer. Some other technologies that I used for this application were Node.js, Webpack, and Sass. Go ahead and try it out!
