EasyListMaker.com

This is a list making app that was initially just PHP/MySQL based. Then I decided to learn how to use MVC Frameworks and I transformed the app into a fully functional Laravel site using Bootstrap for the Front-End. The site uses user authentication and allows the user to create lists and list items. There is complete CRUD capabilities to all levels of the app as well as the use of Polymorphic Relations allowing the user to share their lists with other users if they choose. I also implemented custom route protection using Middleware.Check it out and see how well it works!
